Collins Programier hilfe für Anfänger :)

Willkommen im Geheimen Programier Projekt hier werde ich dir alles bei bringen was du an HTML/CSS wissen musst also bleib dran

1. Installiere dir VS Code ( virsual studio code ) über google

2. erstelle dir einen leeren ordner und füge ihn ein

3. Klicke auf New File und nenne sie index.html das index kannst du durch alles ersetzten was du willst

4. Schreibe Html:5 und dann Enter um das grund gerüst zu bekommen

5. jetzt schreibe h1 und zwischen die klammern Hello world

6. Nun Gehst du auf das zeichen mit den 3 vierecken und einem karo und suchst nach Live Server

7. jetzt installierst du es bitte

8. wenn die installation erfolgreich abgeschlossen ist gehe zurück auf deine seite und mache einen rechtsklick mit deiner maus auf die Datei

9. jetzt klickst du auf open with live Server

10. und schon öffnet sich dein erstes kleines programm viel spaß damit
